# NexusHack 2025 — Event Registration Platform

## 🚀 Live Demo
https://v0-nexushack-2025-website.vercel.app/

## 🎥 Video Demo  


## 🛠️ Tech Stack
- Next.js 14 + Tailwind CSS
- Framer Motion (animations)
- Supabase (database)
- Vercel (deployment)

## ✨ Features
- Real-time registration with Supabase
- Unique Registration ID generation (NXH-2025-XXXX-XXXX)
- Animated confirmation page with video popup
- Duplicate email detection
- Fully responsive

## 🔧 Run Locally
npm install
npm run dev
